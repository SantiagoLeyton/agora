# Retratos de paciente (opcional)

El simulador usa retratos vectoriales clínicos en `ClinicalPatientPortrait`.
Para sustituirlos por fotografías o ilustraciones fotorrealistas con licencia:

1. Coloca imágenes por caso y expresión, por ejemplo:
   - `case-anxiety-001/neutral.webp`
   - `case-anxiety-001/anxious.webp`
2. Expresiones: `neutral`, `sad`, `relieved`, `anxious`, `tense`, `alert`
3. Usa el mismo encuadre (rostro, fondo neutro de consultorio) en todas.

Mantén coherencia de identidad (misma persona por caso).
